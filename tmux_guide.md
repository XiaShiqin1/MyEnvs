# Tmux 极简入门指南 (基于 MyEnvs 定制版)

## 1. 什么是 Tmux？为什么要用它？
Tmux 是一款终端复用工具。它可以帮你解决命令行开发中的两大痛点：
1. **分屏大师**：在一个终端窗口里，你可以同时切分出无数个小窗格，一边看代码、一边跑服务、一边看日志。
2. **永远不掉线**：如果你正在跑一个耗时很久的任务（比如下载或训练模型），如果不小心关掉了终端，或者 SSH 网络断开了，任务就挂了。但如果运行在 Tmux 里，你关掉电脑，任务依然会在后台继续跑，下次连上依然能看到它原封不动的样子。

## 2. 核心概念 (一分钟看懂)
- **Session (会话)**：相当于一个大项目的工作区。即使你彻底关闭终端 App，Session 也会在后台继续活着。
- **Window (窗口)**：相当于浏览器的标签页（Tab）。
- **Pane (窗格)**：在同一个屏幕里被切分成一块块的小格子。
- **Prefix (前缀键)**：**最重要的按键！** Tmux 的绝大部分快捷键都需要先按一次“前缀键”来激活，然后再按功能键。
  👉 Tmux 的默认前缀键是：`Ctrl + b` （**操作方式：先按住 Ctrl 敲一下 b，然后彻底松开双手，再去按后面的键**）。

## 3. 针对 MyEnvs 优化的特色配置
MyEnvs 的 `prepare_all.sh` 脚本已经在原生 Tmux 的基础上为你做了一系列极客级别的优化（主要打通了 macOS 剪贴板和 Vim 习惯），你现在可以直接无缝使用以下特性：

### ✨ 特性 1：鼠标全支持 (新手福音)
原生的 Tmux 必须全靠键盘操作，但我们为你**默认开启了鼠标模式**：
*   **切换窗格**：直接用鼠标点击不同的 Pane 即可激活。
*   **调整大小**：直接用鼠标按住 Pane 之间的分割线拖拽。
*   **查看历史**：直接在 Pane 里用鼠标滚轮上下滚动，即可查看已经刷过去的终端输出（按 `q` 退出滚动状态）。

### ✨ 特性 2：Vim 风格的按键分屏切换
当你的双手在键盘上不想动鼠标时，我们为你绑定了经典的 Vim 方向键。只需按下前缀键 `Ctrl + b`，松开后按：
*   `h`：光标向左切换窗格
*   `j`：光标向下切换窗格
*   `k`：光标向上切换窗格
*   `l`：光标向右切换窗格

### ✨ 特性 3：丝滑的 macOS 全局剪贴板同步
在 Tmux 中复制文本一直是个痛点，但现在它和 macOS 系统剪贴板完全打通了：
*   **方案 A (纯鼠标流)**：直接用鼠标长按并拖拽选中文本，**在你松开鼠标左键的一瞬间**，这段文本就已经自动复制到你的 Mac 系统剪贴板了！（你可以直接切到微信或浏览器用 `Cmd+V` 粘贴）。
*   **方案 B (纯键盘流 / Vim 复制模式)**：
    1. 按 `Ctrl + b` 然后按 `[` 进入复制模式（此时右上角会出现行号指示器）。
    2. 使用方向键或 Vim 的 j/k/h/l 移动光标。
    3. 按 `v` 开始选择文本。
    4. 选好后，按 `y` 或 `Enter(回车)` 复制并自动退出模式。选中的文本将直接进入你的 Mac 剪贴板！

## 4. 日常高频操作速查表

> ⚠️ 记住：下面所有的 `Ctrl+b` 意思是：先按住 Ctrl 敲 b，**彻底松手**，再按后面的键。

### 🚀 会话管理 (在普通终端界面执行)
*   `tmux`：新建一个默认会话并进入。
*   `tmux new -s work`：新建一个名字叫 `work` 的会话。
*   `tmux a`：直接恢复（Attach）回到上一次运行的 Tmux 会话中（比如你刚不小心关掉了窗口，敲这个瞬间恢复）。
*   `tmux a -t work`：恢复指定名字的会话。
*   `tmux ls`：查看当前后台跑着哪些会话。

### 🪟 窗格切分与排版 (Pane 内执行)
*   `Ctrl+b` 然后 `%`：左右垂直分屏。
*   `Ctrl+b` 然后 `"`：上下水平分屏。
*   `Ctrl+b` 然后 `Space (空格)`：**一键切换排版布局**（如果你把屏幕切得很乱，按这个会轮流切换成“平铺、水平等宽、垂直等宽、主次窗口”等几种整齐的自带布局）。
*   `Ctrl+b` 然后 `x`：关闭当前选中的小窗格（底部会提示 y/n，按 y 确认）。
*   `Ctrl+b` 然后 `z`：把当前小窗格临时最大化全屏（Zoom），再按一次可还原回来。

### 📄 标签页管理 (Window 内执行)
*   `Ctrl+b` 然后 `c`：创建一个新的标签页 (Create)。
*   `Ctrl+b` 然后 `n`：切换到下一个标签页 (Next)。
*   `Ctrl+b` 然后 `p`：切换到上一个标签页 (Prev)。
*   `Ctrl+b` 然后 `&`：关闭当前整个标签页（包含里面的所有窗格）。

### 🔌 暂时离开 (Detach)
*   `Ctrl+b` 然后 `d`：暂时离开当前的 Tmux 界面，回到普通的终端环境中。
    *   **魔法在于**：此时你在 Tmux 里跑的任务依然在后台安静地运行！之后你可以随时用 `tmux a` 命令重新接管它。
